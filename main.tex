\section{Test setup}

The laptop I used for testing is a M3 Macbook Air running macOS 26.2.
For creating the test plans and perfoming the perfomance tests I have used Apache JMeter.

\section{Test Plan}
The System Under Test I decided to use for my tests was a sample test website provided by BlazeMeter: https://blazedemo.com/.
In my tests I have targeted the homepage, the reserve page, and the purchase page. The URL for the pages are the following:
\begin{itemize}
  \item Homepage: https://blazedemo.com/index.php
  \item Reserve page: https://blazedemo.com/reserve.php
  \item Purchase page: https://blazedemo.com/purchase.php
\end{itemize}


\section{Types of tests conducted}

The site was tested with three distintive types of perfomance tests:

\vspace{12pt}
{\renewcommand{\arraystretch}{1.5}

\section{Initial Expectations}

Before running the tests I expected the site to respond within 500 - 1000 milliseconds during the baseline perfomance test. I also expected the stress test, and spike tests both to cause about a 100 milliseconds increase in the response times.
